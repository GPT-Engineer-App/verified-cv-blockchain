# verified-cv-blockchain

A blockchain-based CV verification platform tailored for clinicians and healthcare providers. Users can enter their NPI number to create a verified CV, streamlining the verification process for professionals like doctors, nurse practitioners, and physician assistants. The platform also features job postings and allows employers to search for qualified candidates with blockchain-verified qualifications. It integrates the functionality of platforms like Doximity and LinkedIn, focusing on a faster, more secure verification process.
